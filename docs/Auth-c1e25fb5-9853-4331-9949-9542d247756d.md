# Auth

This is the docs for user auth

# User Register

Api End Point

> Type: Post
localhost:5000/api/auth/register

---

### Send

    {
    	"username": "user username",
    	"password": "user password",
    	"email": "user email",
    	"imageUrl": "user image", // Not Required
    }

### Expected Return

Status: 201

    {
    	"token": "user auth token"
    }
