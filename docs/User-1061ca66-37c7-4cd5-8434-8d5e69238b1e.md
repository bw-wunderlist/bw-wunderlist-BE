# User

This is the docs for user

# Get user data

Api End Point

> Type: Get
/api/user/

---

### Send in header

    {
    	"Authorization": "Stored Token"
    }

### Expected Return

Status: 200

    {
    	"username": "user username",
    	"email": "user email",
    	"imageUrl": "user image"
    }

# Update user data

Api End Point

> Type: Put
/api/user/

---
